## 1. サービス概要

本サービスは、

**フリーランスと企業をマッチングし、案件の募集・応募・スカウト・チャットでのやり取りを行える Web プラットフォーム**である。

**「案件を出す → 応募またはスカウト → 会話が成立する」**

という **最小限かつ本質的な価値提供**を目的とした MVP（最小実用製品）として設計している。

---

## 2. 想定するユーザー

### フリーランス側

- IT・Web・デザイン等のスキルを持つ個人
- 副業・業務委託・短中期案件を探している
- 企業と直接コミュニケーションを取りながら仕事を進めたい

### 企業側

- フリーランス人材を探している企業・担当者
- 正社員採用ではなく、柔軟な業務委託を検討している
- スキルや希望条件を見ながらスカウトしたい

---

## 3. 実装する主な機能

### 会員登録・ログイン

- フリーランス・企業それぞれがアカウントを作成できる
- 共通のログイン画面から利用開始
- ログイン後は、ユーザー種別に応じた画面へ自動遷移

---

### プロフィール登録・管理

### フリーランス

- 職種、スキル、自己紹介、稼働条件、希望単価などを登録
- 登録した情報は企業側の検索・判断材料として使用される

### 企業

- 企業名、会社概要、担当者情報などを登録
- フリーランス側が企業情報を確認できる

---

### 案件管理（企業側）

- 企業は案件を作成・編集・公開できる
- 公開中の案件のみがフリーランス側に表示される
- 下書き・停止状態の案件は検索結果に表示されない

---

### 案件検索・閲覧（フリーランス側）

- 公開されている案件を一覧で確認できる
- キーワードや報酬条件で検索可能
- 案件詳細ページで仕事内容や条件を確認できる

---

### 応募・スカウト

- フリーランスは案件に応募メッセージを送信できる
- 企業はフリーランスに対してスカウトメッセージを送信できる
- 応募・スカウトが発生した時点で、チャットルームが自動生成される

---

### チャット機能

- 応募・スカウト後は、チャット画面で直接やり取りできる
- メッセージ送信・削除が可能
- 未読状態はスレッド単位で管理され、画面上で確認できる

---

## 4. ユーザーの基本的な利用の流れ

### フリーランス側の流れ

1. 会員登録・ログインを行う
2. プロフィール情報を登録する
3. 案件一覧から興味のある案件を探す
4. 案件詳細を確認し、応募メッセージを送信する
5. 自動生成されたチャット画面で企業とやり取りを行う

---

### 企業側の流れ

1. 会員登録・ログインを行う
2. 企業プロフィールを登録する
3. 案件を作成し、公開する
4. 応募された案件やフリーランス一覧を確認する
5. 応募対応またはスカウトを行い、チャットでやり取りを行う

## **5. 利用する画面一覧（利用者別）**

---

## 5-1. フリーランス・企業 共通で利用する画面

| 画面名 | 内容 |
| --- | --- |
| 共通ログイン画面 | フリーランス・企業共通のログインおよび新規登録導線を提供する画面 |
| パスワードリセット画面 | 登録済みメールアドレス宛にパスワード再設定用リンクを送信する画面 |
| チャット画面 | 応募またはスカウト後に、メッセージの送受信を行う画面（利用者により表示内容が異なる） |

---

## 5-2. フリーランス専用画面

| 画面名 | 内容 |
| --- | --- |
| フリーランス登録画面 | フリーランスのアカウント作成およびプロフィール情報を登録する画面 |
| フリーランス案件一覧画面 | 公開中の案件を一覧で表示し、検索・閲覧を行う画面 |
| 案件詳細画面 | 案件の詳細内容を確認し、応募を行う画面 |
| 応募メッセージ入力画面 | 案件応募時に企業へ送信するメッセージを入力する画面 |
| 応募案件一覧画面 | 応募した案件を一覧で確認できる画面（応募中／終了タブ切替） |
| スカウト一覧画面 | 企業から送られたスカウトを一覧で確認する画面 |
| フリーランスプロフィール設定画面 | フリーランス自身のプロフィール情報を編集・更新する画面 |

---

## 5-3. 企業専用画面

| 画面名 | 内容 |
| --- | --- |
| 企業登録画面 | 企業アカウントの作成および企業プロフィールを登録する画面 |
| 企業フリーランス一覧画面 | フリーランスを検索・一覧表示し、詳細を確認できる画面 |
| スカウト入力画面 | フリーランスに対してスカウトメッセージを送信する画面 |
| 応募案件一覧画面（企業側） | フリーランスから応募された案件を一覧で確認する画面 |
| 企業案件一覧画面 | 企業が自社案件を一覧で管理する画面 |
| 企業案件登録画面 | 新しい案件を登録する画面 |
| 企業案件編集画面 | 既存案件を編集・更新する画面 |
| 企業プロフィール設定画面 | 企業情報を編集・更新する画面 |
